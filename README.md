# Test application



## Installation

1. Clone the above project.

```bash
git clone https://github.com/Developer2250/test-application.git
```

2. Install Composer and Node module.
```bash
composer install
```

```bash
npm install 
```

3. Run the above project. 
```bash
php artisan serve
```
